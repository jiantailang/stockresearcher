# Financial Data Dashboard

StreamlitとYahoo Finance APIを活用したインタラクティブな金融データ可視化ダッシュボード

##  概要・開発の背景
このプロジェクトは、自分自身の株式投資において、データに基づく確かなエビデンス（根拠）を持たせるために作成しました。
株価などの金融データの収集と可視化を自動化し、直感的な市場分析を可能にすることで、より客観的で納得感のある投資判断をサポートします。

##  特徴
- `yfinance` を利用したリアルタイムな金融データの取得・連携。
- `pandas` を活用した、取得データの柔軟な前処理および分析機能。
- `streamlit` を用いた、分かりやすくインタラクティブなWebダッシュボードUIの提供。

## 🛠 使用技術
- Python 3.x
- Streamlit
- yfinance
- Pandas

##  セットアップ
1. `git clone https://github.com/user/repo.git`
2. `pip install -r requirements.txt`
3. `streamlit run app.py`
